# mrgarrido.github.io

Personal academic website of **Klaus Garrido Tenorio**, PhD candidate at the Rotterdam School of Management, Erasmus University Rotterdam.

Built on the [Academic Pages](https://github.com/academicpages/academicpages.github.io) Jekyll template and hosted on GitHub Pages. 

## Editing the site

- `_config.yml` — name, bio line, avatar, social links
- `_pages/about.md` — homepage (bio, research, presentations)
- `_pages/teaching.md`, `_pages/awards.md` — subpages
- `_data/navigation.yml` — top navigation bar
- `files/Klaus_Garrido_Tenorio_CV.pdf` — the downloadable CV (replace with a new export to update)
- `images/profile.png` — profile photo shown in the sidebar

## Local preview (optional)

GitHub Pages builds the site automatically on every push, so no local setup is required. To preview locally:

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000.
