This is the negotiation interface from my behavioral experiments on human–AI negotiation (see the technical paper). You play the Supplier; the Retailer is an AI agent. Its decisions — when to accept and what to counter-offer — come from the experiment's rule-based engine; its chat messages can be generated by a free LLM (see Settings below). Chat is non-binding: only offers sent through the interface table, or confirmed there, are binding. No data is collected on this page.

The Production Cost (PC) is 1.00€. Wholesale price must be between 1–5€, quantity between 1–100 units. Demand is uniform on [0, 100].
The Retailer chose not to disclose their Retail Price (RP).

This page is fully static (GitHub Pages), so there is no server to hold an API secret. To see the LLM-powered version, paste your own free OpenRouter API key — it is stored only in your browser (localStorage) and sent only to OpenRouter. Without a key, the demo uses the experiment's scripted messages; the negotiation logic is identical either way. Incoming messages are screened with nvidia/nemotron-3.5-content-safety:free, and replies are written by the chat model below.
